# SecOpsHub 产品愿景

> 版本: v0.1 | 创建日期: 2026-01-18 | 状态: 草稿

## 1. 愿景

**打造一站式安全运营中枢**，整合攻击模拟验证、AI辅助决策、知识管理等能力，帮助安全团队从被动响应转向主动防御。

## 2. 产品矩阵

```
SecOpsHub
├── AttackProbe (BAS平台)        # 攻击模拟与有效性验证
│   ├── 传统安全BAS引擎
│   └── LLM安全BAS引擎
│
├── LLM-Security (独立模块)      # LLM大模型安全测试
│   └── 可独立部署，也可作为BAS组件
│
└── AI-Brain (知识管理)          # 安全团队知识管理
    ├── brain-search             # 全文搜索
    ├── brain-ask                # RAG问答
    ├── brain-archive            # 对话归档
    └── brain-decision           # ADR管理
```

## 3. 目标用户

| 产品 | 主要用户 | 核心价值 |
|------|---------|---------|
| AttackProbe | 安全运营/红队 | 持续验证防护有效性 |
| LLM-Security | AI安全工程师 | 快速评估LLM安全风险 |
| AI-Brain | 安全团队全员 | 沉淀知识，提升效率 |

## 4. 发展路径

1. **Phase 1 (当前)**: LLM-Security MVP 已完成，AI-Brain 基础功能已有
2. **Phase 2**: 完善 AttackProbe BAS 平台，整合 LLM-Security
3. **Phase 3**: 引入传统 BAS 能力，完整 ATT&CK 映射
4. **Phase 4**: 多产品联动，形成安全运营闭环
