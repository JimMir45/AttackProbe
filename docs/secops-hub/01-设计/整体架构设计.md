## 2. 模块关系

| 模块 | 类型 | 部署方式 | 依赖关系 |
|------|------|---------|---------|
| AttackProbe | 主平台 | 独立部署 | 可集成 LLM-Security |
| LLM-Security | 独立模块 | 独立/嵌入 | 无外部依赖 |
| AI-Brain | 工具集 | CLI工具 | 需要本地知识库 |

## 3. 技术栈统一

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 后端语言 | Go 1.22+ | 统一使用 Go |
| Web框架 | Gin | 轻量高效 |
| 前端框架 | Vue 3 + Element Plus | 统一UI风格 |
| 数据库 | SQLite → PostgreSQL | 渐进式升级 |
| 部署 | 单二进制 | 简化运维 |
